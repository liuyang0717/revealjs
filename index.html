Worm-like Chain

reporter: liuyang

Introduction

  • The worm-like chain (WLC) model in polymer physics is used to describe the behavior of polymers that are semi-flexible.
  • The WLC model is the continuous version of the Kratky-Porod model.
  • Beyond that, an ideal chain (or freely-jointed chain) is the simplest model to describe polymers.

Freely-jointed Chain Model

Feature

  • a random walk
  • neglects any kind of interactions among monomers

model elements

Entropic elasticity of an ideal chain

Worm-like Chain Model

Feature

  • fairly stiff
  • successive segments pointing in roughly the same direction
  • persistence length within a few orders of magnitude of the polymer length
